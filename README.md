## What is TFB / TCF?

**Theory of Fundamental Belief (TFB)** is a theoretical framework that defines **belief** as the foundational cognitive axis underlying human perception, behavior, and decision-making. The theory proposes that belief is not merely an opinion, emotion, or cultural construct, but a **structural cognitive mechanism** that precedes interpretation, choice, and action across psychological, philosophical, and cognitive domains.

The applied technical layer is the **Anonymo AI Framework 24x24** — a governance and observation middleware between humans and AI systems, built on TCF/TFB principles.

## Audio Capture System — Research Proof

This repository includes documentation related to the audio capture and frequency analysis system, developed as part of the Anonymo AI Framework research. The system uses the Web Audio API for real-time voice pattern capture and visualization, including dual-sensor configuration (phone + wristwatch).

The audio capture system is documented here as **proof of research and empirical testing**. It demonstrates the practical application of the Audio Triangle concept (Voice vertex, Physiology vertex, Pattern vertex) described in the framework.
