Seção 01

Visão Geral do Projeto

O Anonymo AI é um sistema de captura e análise de frequência de áudio em tempo real, desenvolvido como parte do framework de pesquisa TCF/TFB. O sistema foi projetado para capturar, visualizar e anotar sinais de áudio de voz e fisiológicos por meio de uma interface baseada em navegador, utilizando microfones profissionais e sensores externos.

A fase atual concentra-se na análise de áudio de voz — capturando frequência vocal, amplitude e padrões espectrais em tempo real — com integração futura planejada para sensores fisiológicos externos (baseados no pulso/punho).

"Este sistema opera na interseção entre análise de voz, monitoramento fisiológico e documentação estruturada de pesquisa — projetado para precisão, reprodutibilidade e observação em múltiplos ângulos."
Seção 02

Contexto de Pesquisa e Autorização

Todas as sessões conduzidas com este sistema são realizadas mediante consentimento informado, em conformidade com os padrões éticos de pesquisa aplicáveis. Os participantes são informados sobre a natureza da coleta de dados antes do início de qualquer sessão.

Os dados coletados são utilizados exclusivamente para fins de pesquisa no âmbito do framework Anonymo AI. Nenhum dado é compartilhado publicamente sem autorização explícita. O pesquisador possui credenciais profissionais adequadas para conduzir este tipo de pesquisa observacional.

Todas as sessões de pesquisa requerem termo de consentimento livre e esclarecido assinado pelos participantes, conforme legislação vigente.
Seção 03

Arquitetura de Sensores — O Triângulo de Áudio

O modelo central de observação é construído em torno do que o framework denomina "Triângulo de Áudio" — uma configuração multi-sensor que capta sinais de áudio de diferentes posições simultaneamente, permitindo análise cruzada que elimina variáveis de distância e fornece dados mais ricos.

Posição do Sensor Tipo de Captura Variável Principal
Lapela / Proximidade da boca Voz — captura direta Frequência vocal, articulação
Pulso / Sensor fisiológico Fisiológico — indireto Ressonância corporal, estado fisiológico
Externo / Ambiente Ruído ambiental Classificação de interferência de fundo

Nota metodológica importante: quando o sensor de pulso é o dispositivo de captura primário e o telefone/microfone está a uma distância maior, a interpretação lógica se inverte — a variável de distância se desloca do pulso para a fonte de voz. O sistema considera essa inversão em todas as variantes de observação.

Dentro deste modelo de observação triangular, o framework atual mapeia múltiplas variantes de observação — cada uma representando uma combinação distinta de posições de sensores, distâncias e prioridades de sinal. A matriz completa de variantes é documentada separadamente dentro do framework TCF/TFB.

Seção 05

Framework Colaborativo de IA

Este projeto de pesquisa emprega um modelo de colaboração distribuída com IA, onde diferentes sistemas de inteligência artificial são designados para domínios específicos do trabalho:

Plataforma Domínio
Genspark (Spark) Sistema de análise de áudio — desenvolvimento da interface, visualização de frequência
Manus Fisiologia — integração do sensor de pulso, análise de sinais corporais
GPT (OpenAI) Documentação de pesquisa, assistência de escrita, framework teórico
Gemini (Google) Estudos de campo, cruzamento de dados, pesquisa aplicada

Todas as contribuições das IAs são de natureza assistiva. O framework conceitual, o design da pesquisa, as decisões estruturais e a interpretação dos resultados permanecem como trabalho intelectual exclusivo do autor humano.

Seção 06

Nota Metodológica

Processo de Pesquisa Híbrido:
Este trabalho foi desenvolvido por meio de um processo de pesquisa híbrido. Ferramentas de inteligência artificial foram utilizadas como assistência de escrita e rascunho nas etapas posteriores da documentação, enquanto o framework conceitual, o design estrutural e as ideias centrais foram desenvolvidos pelo autor humano.

O uso de múltiplas plataformas de IA neste projeto reflete uma escolha metodológica deliberada para aproveitar capacidades especializadas em diferentes domínios de pesquisa, mantendo autoria unificada e responsabilidade intelectual.
