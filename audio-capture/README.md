## Description

Real-time audio capture and frequency analysis system developed as part of the Anonymo AI Framework 24x24 research. This system demonstrates the practical application of the **Audio Triangle** concept — the three-vertex observation model (Voice, Physiology, Pattern) that forms the core of the framework's multi-sensor triangulation approach.

The system uses the Web Audio API for voice pattern capture and visualization, including support for dual-sensor configuration (phone microphone + wristwatch sensor) and professional USB microphones.

---

## Research Context

This system is part of the empirical testing that validates the framework's observation angles. Each vertex of the Audio Triangle contains multiple angles of interpretation — a number that continues to grow with each empirical discovery. The environment is not a separate vertex but one of the observation angles that exists within each point of the triangle.

The audio capture API and related systems are also deposited on other platforms (including Cloudflare and GenSpark) as additional proof of research.
